# ⚖️ Lexia & Asociados - Sitio Web

Sitio web estático del despacho **Lexia & Asociados**, enfocado en servicios legales en México: derecho civil, familiar, laboral, penal, mercantil y asesoría empresarial.

## ✨ Características

- 🧭 Navegación principal con menú de servicios y versión móvil.
- 📄 Páginas dedicadas para áreas de práctica legal.
- 📰 Sección de blog con artículos temáticos.
- 📞 Página de contacto para agendar consulta.
- 🎨 Estilos personalizados y diseño responsive.

## 📁 Estructura del proyecto

```text
.
├── index.html
├── nosotros.html
├── servicios.html
├── blog.html
├── contacto.html
├── aviso-privacidad.html
├── terminos-condiciones.html
├── servicios/
└── blogs/
```

## 🚀 Uso local

Como es un sitio estático, puedes abrir `index.html` directamente en tu navegador o levantar un servidor local:

```bash
python3 -m http.server 8000
```

Luego visita: `http://localhost:8000`
